# Source overlap and gap analysis

## Stato

Report **strutturale** — da popolare con ingest reali.

## Metriche da calcolare per ogni regione

| Metrica | Descrizione |
|---------|-------------|
| multi_source | Record in ≥2 fonti ufficiali |
| single_source | Solo 1 fonte |
| new_from_source | Unici aggiunti da ciascuna fonte |
| province_under | Province sotto mediana |
| category_under | Categorie sottorappresentate |
| failed_sources | Fonti con failures > 0 |
